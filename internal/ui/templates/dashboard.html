How to configure PAC
  • macOS Follow the Automatic Proxy Configuration instructions in System Settings and paste the URL above. Safari, Chrome, and most other browsers automatically use these system-wide settings.
  • Windows In Use a proxy server in Windows, follow the To set up a proxy server using a setup script section and paste the URL above. Chrome, Edge, and most browsers automatically use these system-wide settings.
  • Firefox Firefox manages its own proxy settings independently of the OS. See Connection settings in Firefox — select Automatic proxy configuration URL and paste the URL above.
Troubleshooting PAC issues

Most browsers cache PAC files for multiple hours. Known solutions:

  • Chrome Chrome always uses system-level PAC settings. To force a refresh: paste chrome://net-internals/#proxy into the address bar and click Re-apply settings.
  • Safari & other browsers using macOS system-wide PAC In macOS proxy settings, disable Automatic Proxy Configuration and save, then re-enable it and save again.

For other browsers or operating systems there is no guaranteed fix — looking up browser-specific instructions may help.
